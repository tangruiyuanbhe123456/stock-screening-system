# 四阶段选股诊股系统

一个结构化的 A 股选股与持仓监控系统，分四个阶段执行。每个阶段有明确的输入、输出和决策标准。

---

## 阶段一：月度初筛 — A股咽喉卡点拆解

**频率：** 每月一次（每月 1-5 日）
**目标：** 从全市场锁定 10-15 只符合结构条件的候选股

### 步骤

1. **逆向拆解赛道 BOM 清单**
   - 选择当前政策强驱动的赛道（半导体、高端装备、工业母机、航空发动机、医疗器械、新材料等）
   - 拆解该赛道从原材料→零部件→设备→系统的完整 BOM 链路
   - 标注每个环节的国产化率（<5% / 5-30% / >30%）

2. **筛选条件（硬性门槛）**

   | 条件 | 要求 | 数据来源 |
   |------|------|---------|
   | 国产化率 | < 5% | 券商研报、产业调研 |
   | 扩产周期 | > 18 个月 | 公司公告、产业经验 |
   | 市场地位 | 垄断或寡头（国内前 3） | 行业报告 |
   | 企业标签 | 专精特新 / 单项冠军 / 小巨人 | 工信部名单 |
   | 市值 | 30 - 150 亿 | 实时行情 |
   | 行业 | 卡脖子方向（半导体/军工/高端制造/材料/医疗设备） | — |

3. **输出表格**

   ```
   | 股票 | 代码 | 赛道 | 卡脖子环节 | BOM位置 | 国产化率 | 扩产周期 | 市值(亿) | 标签 | 初筛结论 |
   |------|------|------|-----------|---------|---------|---------|---------|------|---------|
   | XXX  | 000000| 半导体| 前道检测设备 | 设备 | <5% | 24m | 85 | 专精特新 | 通过 |
   ```

4. **数量控制：** 每月初筛保留 ≤15 只

## 阶段二：季度复审 — 财务数据硬核穿透

**频率：** 每季度（财报发布后 2 周内）
**目标：** 从初筛池中筛选出 3-5 只财务质量过关的股票

### 步骤

1. **审计近 4 年财报**
   - 提取并分析以下财务指标：

   | 维度 | 指标 | 检查要点 |
   |------|------|---------|
   | 营收质量 | 主管业务收入 CAGR | 连续 3 年正增长 |
   | 毛利 | 主管业务驱动的毛利率变化 | 毛利率稳定或提升，排除靠非主业涨利润的 |
   | 资本开支 | CapEx 趋势 | 近 2 年 CapEx 明显爬坡 |
   | 在建工程 | 在建工程余额 | 秘密爬坡信号（不显眼但持续增加） |
   | 现金流 | 经营性现金流 / 营收 | > 0.5 为健康 |
   | 研发投入 | 研发费用率 | ≥ 8% 为合格，≥ 15% 为优秀 |
   | 负债 | 资产负债率 | < 60%（制造业标准） |

2. **检查定增投向**
   - 查询近 3 年定增 / 增发记录
   - 定增资金是否投向卡脖子环节（是 → 加分，否 → 中性）
   - 定增对象是否有产业资本/国家大基金（有 → 强烈加分）

3. **排除规则（硬性门槛）**
   - **剔除** 近 3 个月机构研报 > 15 篇的股票（太拥挤，预期已打满）
   - **剔除** 毛利率连续 3 年下滑的股票
   - **剔除** 营收增速 < 10% 且无合理解释的股票

4. **输出表格**

   ```
   | 股票 | 代码 | 营收CAGR | 毛利率趋势 | CapEx趋势 | 在建工程 | 现金流比 | 定增投向 | 研报篇数 | 复审结论 |
   |------|------|---------|-----------|----------|---------|---------|---------|---------|---------|
   | XXX  | 000000| 25%     | 稳定↑     | 爬坡↑    | 增长↑   | 0.7     | 卡脖子  | 8       | 通过    |
   ```

## 阶段三：买前终审 — AI 红队对抗伪证

**执行时机：** 准备买入前最后一步
**目标：** 扮演极度苛刻的空头，用对抗性思维审查标的

### 检查清单

你必须扮演一个**极度苛刻的空头分析师**，对以下每个维度进行对抗性审查：

#### 1️⃣ 大客户风险
- [ ] 前 3 大客户占比是否 > 50%？（单一客户依赖）
- [ ] 大客户是否有自研计划？（例如：客户公告过自研团队、招聘自研岗位）
- [ ] 大客户是否有引入二供的计划？（查招标信息、环评报告、供应链新闻）
- [ ] 如果大客户丢失，公司是否有替代客户？

#### 2️⃣ 技术替代风险
- [ ] 公司技术路径未来 18 个月是否会被替代？
- [ ] 是否有新的技术路线正在实验室/中试阶段（固态→液态、AI→传统、非侵入→侵入）
- [ ] 同行的技术迭代速度是否更快？
- [ ] 公司是否有专利护城河？核心专利到期时间？

#### 3️⃣ 价格战风险
- [ ] 竞争对手是否有扩产计划？
- [ ] 行业是否处于供过于求的拐点？
- [ ] 毛利率是否处于历史高位（下行风险 > 上行空间）？
- [ ] 是否有新进入者（跨界玩家）威胁？

#### 4️⃣ 政策风险
- [ ] 公司是否依赖单一补贴政策？
- [ ] 补贴是否即将退坡/取消？
- [ ] 地缘政治是否可能切断供应链关键节点？
- [ ] 被美国/欧盟制裁清单的风险？

### 输出格式

```
===== AI 红队对抗报告 =====
股票：XXX（000000）

【大客户风险】⭐️⭐️⭐️ (3/5)
- 前3大客户占比：45%
- 大客户自研风险：低（客户无自研公告）
- 二供引入风险：中（客户已发二供招标）
- 评价：可控，但需跟踪二供进展

【技术替代风险】⭐️⭐️⭐️⭐️ (4/5)
- 18个月替代风险：低
- 专利护城河：中等（核心专利2028年到期）
- 同行速度：相对领先
- 评价：技术壁垒有效

【价格战风险】⭐️⭐️ (2/5)
- 竞争对手扩产：有（XX产能翻倍）
- 供过于求风险：高（行业产能利用率<70%）
- 评价：⚠️ 主要风险点

【政策风险】⭐️⭐️⭐️ (3/5)
- 补贴依赖：中等
- 地缘风险：低

⚠️ 终审结论：通过 / 暂缓 / 否决
⚠️ 核心风险：_________________
```

## 阶段四：持股监控 — 贝叶斯动态熔断

**频率：** 每个月更新一次
**目标：** 持续跟踪持仓股的关键里程碑，未达标则触发熔断

### 规则

1. **为每只持仓股列出未来 2 个季度的 3 个硬核里程碑**
   - 里程碑必须是可验证的（打样、投产、客户验证、订单公告、环评获批等）
   - 每个里程碑设最晚时限
   - 通过 互动易（深交所）或 e互动（上交所）确认进展

2. **熔断机制**
   - 若在 互动易/e互动 上未确认到里程碑进展 → 标记为 ❌
   - 若超时未达成 → 启动熔断（减仓 1/3）
   - 若连续 2 个里程碑失败 → 清仓

3. **输出表格**

   ```
   | 股票 | 代码 | 里程碑 | 最晚时限 | 互动易确认 | 状态 | 熔断阈值 |
   |------|------|--------|---------|-----------|------|---------|
   | XXX  | 000000| 关键客户打样完成 | 2026-08-31 | [链接/截图] | 待确认 | 7日未确认→减1/3 |
   | XXX  | 000000| 定增募投项目投产 | 2026-10-15 | [链接/截图] | 进行中 | 超时→减1/3 |
   | XXX  | 000000| 新产品通过验证 | 2026-12-31 | [链接/截图] | 未到 | — |
   ```

4. **每月更新**
   - 查询互动易/e互动 最新回复
   - 更新里程碑状态
   - 检查是否有新的公告（业绩预告、重大合同、解禁等）
   - 重新评估 3 个里程碑是否仍然有效

## 执行流程（自动化 cron 建议）

```
每月 1-5日 → 阶段一：月度初筛
季报后 2周内 → 阶段二：季度复审
买入前 → 阶段三：买前终审
每月 15日 → 阶段四：持股监控
```

## 核心理念

> 四阶段量化赛选，每个阶段设硬性门槛，通过的才能进入下一阶段。
> 红队对抗 = 自己先做空自己，找到所有可能的死法。
> 贝叶斯熔断 = 每次新信息更新先验概率，不达预期就行动。
