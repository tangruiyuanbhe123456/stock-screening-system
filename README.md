# stock-screening-system

**四阶段选股诊股系统** — 为 A 股投资者设计的结构化筛选与持仓监控框架。

## 四阶段流程

| 阶段 | 频率 | 目标 |
|------|------|------|
| ① 月度初筛 | 每月 | 筛选国产化率<5%且扩产周期>18个月的垄断环节，专精特新/单项冠军，市值30-150亿 |
| ② 季度复审 | 每季度 | 审计近4年财报，检查CapEx/在建工程爬坡信号，剔除研报>15篇的拥挤标的 |
| ③ 买前终审 | 买入前 | AI红队对抗：扮演空头审查大客户自研/二供风险、技术替代、价格战、政策风险 |
| ④ 持股监控 | 每月 | 设定3个硬核里程碑+最晚时限，通过互动易确认，未达标则贝叶斯熔断减仓 |

## 安装

```bash
npx skills init
npx skills install stock-screening-system
```

或直接查看 SKILL.md 获取完整工作流。
